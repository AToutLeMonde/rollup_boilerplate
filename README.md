# Инструкция

### Установка зависимостей

```
npm install --force
```

Переменные окружения задаются в файлах .env.dev и .env.prod

## `Режим разработки`

```
npm run start
```
Приложение будет доступно по адресу http://localhost:3000/


## `Сборка production `

```
npm run build
```

Все артефакты после сборки располагаются в каталоге build/prod
